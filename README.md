
# XBOIL
XBOIL is a computational framework for the systematic evaluation of heat flux partitioning (HFP) models and associated bubble dynamics sub-models used in nucleate boiling modelling.

The framework was developed to enable large-scale assessment of boiling closure combinations against experimental data.

## Associated publication
The methodology and results obtained using XBoil are described in:

O. Bidar and M. Colombo  
*Evaluations of boiling heat flux partitioning and bubble parameter models using a 0-D framework.*  
International Journal of Heat and Mass Transfer (under review).

## Repository contents
This repository will contain:
- The **XBoil evaluation framework**
- Scripts used to generate the **model combinations and evaluation results**
- Example workflows for testing additional boiling sub-models

## Availability
The repository has been created to support reproducibility of the published study.  
The full code and dataset will be released following publication of the paper.
